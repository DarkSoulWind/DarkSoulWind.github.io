A computer is a machine that processes data

The purpose of a computer is to take data, process it, then output it.

A computer system consists of hardware and software that work together to process data/complete tasks

  • Hardware is the physical stuff that makes up your computer system, like the CPU, motherboard, monitor and printer.
  • Software is the programs or applications that a computer system runs such as the operating syste, a word processor or video game.


Networks also allow computers to connect with each other from any place in the world and share data with each other.
