Open source software is given away with its source code

  • This is a type of software where the source code is made freely available, meaning users may legally modify the code to create their own spin-off software, which can be shared under the same licence and terms as the original software.
  • Popular open source software is always supported by a strong online community of users who actively help to improve the software - anyone can play with the source code and suggest bug fixes and improvements to the original developers.
  • Several examples of open source software include popular web browser Mozilla Firefox, photo editor GIMP and VLC media player.

Pros and cons of open-source software:

Advantages of Open Source Software Disadvantages of Open Source Software
  • It is (usually) free.
  • Made for the greater good, not profit - it benefits everyone, encourages collaboration, sharing of ideas.
  • Software can be adapted by users to fit their needs.
  • Wide pool of collaborators can be more creative and innovative than the programmers of one company.
  • Popular software is very reliable and secure - any problems are quickly solved by the community.
  • Small projects may not get regular updates and so could be buggy or have unpatched security holes.
  • There may be limited user documentation.
  • No warranties if something goes wrong.
  • No customer support, althout community forums will often make up for this.
  • Companies using open-source code to make custom software may not want competitors to see their code, but they have no choice.

Proprietary software is Closed Source Software

  • This is a type of software where only the compiled code is released, and the source code remains a closely-guarded secret. They are usually paid for.
  • Proprietary software licenses restrict the modification, copying and redistribution of the software.
  • This software is often used by businesses, as they tend to have better customer support options.
  • Examples of proprietary software include Microsoft Office and Adobe Creative Cloud.

Pros and cons of proprietary software:

Advantages of Proprietary Software Disadvantages of Proprietary Software
  • Typically comes with warranties, documentation, and customer support.
  • Should be well-tested and reliable as the company's reputation depends on this.
  • Fixes and updates come regularly, while open source will vary more.
  • Usually cheaper for companies than developing their own custom-built software.
  • Can be expensive (Final Cut Pro at $299).
  • Software may not exactly fit a user's needs, and they can't do anything about it.
  • Software companies may not maintain older software after warrranties expire - they'll want people to buy their latest product.

OSes have several functions:

  • Communicate with internal and external hardware via the device drivers.
  • Provide a user interface, for the user to interact with.
  • Provide a platform for different applications to run.
  • Allows multitasking by controlling memory resources.
  • Deal with file management and disk management.
  • Manage the security of the system.

User Interfaces

A User Interface (UI) allows the user to interact with a computer system.

Graphical User Interfaces (GUIs) are the most common type - they're designed to be easy for everyday users by making them visual, interactive and intuitive. They are optimised for specific input methods, typically consisting of windows, icons, menus and pointers (WIMP). GUIs like those of Android and iOS were created for touchscreen devices, using finger gestures like pinching and swiping in place of a mouse.

A command-line interface (CLI) is text-based. The user enters specific commands to complete tasks. They care less resource-heavy compared to GUIs. They aren't, however suitable for everyday users, but for advanced users, they can be far more efficient and powerful than a GUI. They can be used to automate processes using scripts (simple programs).

Other examples of utility software include:

  • Compression software, which permanently or temporarily removes data from files to reduce their size, therefore 'compressing' them.
  • Encryption software, which scrambles data to stop third-parties from accessing it.
  • Disk defragmentor, which helps put broken up files back together to prevent further fragmentation on a hard disk drive.
