The CPU has three main parts:

The Control Unit (CU)

  • The Control Unit's main job is to manage the fetching, decoding and execution of program instructions by following the fetch-decode-execute cycle.
  • It also controls the flow of data inside and outside the CPU.

The Arithmetic Logic Unit (ALU)

  • The Arithmetic Logic Unit's job is to do all the calculations.
  • This can include simple addition and subtraction, comparing the sizes of numbers and multiplications and divisions using repeated addition and subtraction.
  • It can also include logic operations such as AND, OR and NOT nad binary shifts - don't forget that computers only process binary data.

The Cache

  • The cache is very fast memory in the CPU, much faster than RAM.
  • It stores regularly used data so that the CPU can access it quickly the next time it's used.
  • When the CPU requests data, it checks the cache first to see if the data is there. If not, it will fetch it from the RAM.
  • They have a very low capacity and are expensive compared to RAM and secondary sotrage.
  • There are three different levels of cache memory:
    1. L1 is quicket but has the lowest capacity.
    2. L2 is slower than L1 but can hold more.
    3. L3 is the slower, but can hold the most.
